# Services Directory

This directory contains external service integrations for SpecCraft.

## Planned Services

- **cosmosdb.ts** - Azure CosmosDB client and database operations
- **blob-storage.ts** - Azure Blob Storage client for file attachments
- **openai.ts** - Azure OpenAI client for AI assistance features
- **auth.ts** - Azure AD authentication service

Each service will be implemented in subsequent tasks.
