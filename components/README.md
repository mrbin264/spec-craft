# Components Directory

This directory contains reusable React components for SpecCraft.

## Planned Components

- **SpecEditor** - Markdown editor with live preview
- **AIAssistantPanel** - AI assistance interface
- **VersionHistory** - Version history and diff viewer
- **TraceabilityGraph** - Visual traceability graph
- **CommentsPanel** - Commenting and discussion interface
- **WorkflowControls** - Workflow stage transition controls

Each component will be implemented in subsequent tasks.
